## 🎯 快速了解项目

### 项目定位
**企业级大语言模型应用开发与运维平台**

### 核心价值
1. **降低开发门槛**：无需深入了解LLM技术细节，即可构建AI应用
2. **提高开发效率**：可视化工作流设计，快速构建复杂业务逻辑
3. **降低成本**：统一管理多模型，优化Token使用
4. **提升质量**：混合检索、重排序等机制提升应用质量

### 技术特点
- 🏗️ **分层架构**：Handler-Service-Model三层架构
- 🔌 **依赖注入**：使用Injector实现松耦合
- 🔄 **异步处理**：Celery处理耗时任务
- 🗄️ **多数据库**：PostgreSQL + Redis + Weaviate
- 🔍 **混合检索**：FAISS + BM25 + 重排序

## 🏗️ 技术架构

### 后端技术栈
- **框架**：Flask 3.0+
- **ORM**：SQLAlchemy 2.0+
- **数据库**：PostgreSQL、Redis、Weaviate
- **异步**：Celery 5.3+
- **LLM框架**：LangChain 0.2+、LangGraph 0.2+
- **向量检索**：FAISS、Weaviate
- **依赖注入**：Injector 0.21+

### 前端技术栈
- **框架**：Vue 3.4+ + TypeScript 5.4+
- **构建工具**：Vite 5.2+
- **状态管理**：Pinia 2.1+
- **UI组件**：Arco Design 2.55+
- **工作流可视化**：Vue Flow 1.41+
- **数据可视化**：ECharts 5.6+

## 🚀 快速启动（3步）

### 1. 启动Docker服务
```powershell
start-local.bat  # 或手动启动Weaviate
```

### 2. 启动后端
```powershell
cd imooc-llmops-api\imooc-llmops-api-master
.venv\Scripts\Activate.ps1
python -m app.http.app
```

### 3. 启动前端
```powershell
cd imooc-llmops-ui\imooc-llmops-ui-master
npm run dev
```

**访问地址**：
- 前端：http://localhost:5173
- 后端：http://localhost:5000
